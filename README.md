## Para testar/contribuir no Projeto

### Instalando Dependências

> Backend
```bash
cd api/ && npm install
``` 
> Frontend
```bash
cd src/ && npm install
``` 
### Executando aplicação

* Para rodar o back-end:

  ```
  cd api/ && npm start
  ```
* Para rodar o front-end:

  ```
    cd src/ && npm start
  ```

### Executando Testes

* Para rodar todos os testes:

  ```
    npm test
  ```
